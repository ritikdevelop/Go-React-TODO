# Go-React TODO

This is a simple TODO application built with Go and React.

## Features
- Create, read, update, and delete tasks.
- User authentication.
- Responsive design.

## Getting Started

### Prerequisites
- Go 1.x
- Node.js 12.x or higher

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ritikdevelop/Go-React-TODO.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Go-React-TODO
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
